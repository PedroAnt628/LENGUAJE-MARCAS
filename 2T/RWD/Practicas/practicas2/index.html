Responsive Web Design con Mobile First

Este diseño adapta su estructura según el tamaño de la pantalla:

· Móvil (< 768px): Layout vertical, todo en columna · Tablet (768px - 1023px): Grid 2 columnas, menú lateral + contenido/aside apilados · Desktop (≥ 1024px): Grid 3 columnas, todas visibles simultáneamente

Los estilos se cargan en cascada: primero móvil (base), luego tablet, finalmente desktop.

Esta estructura utiliza tres archivos CSS separados en la carpeta - css/:

· movil.css - Estilos base para todos los dispositivos · tablet.css - Estilos adicionales para tablets (≥768px) · desktop.css - Estilos adicionales para escritorio (≥1024px)
